# Knuckle Joint Assembly

## 📌 Overview

This repository contains a **3D CAD model of a Knuckle Joint Assembly** created in Autodesk Fusion 360 and exported as a **STEP (.stp)** file. The model demonstrates the design and assembly of a standard knuckle joint used to connect two rods under tensile load while allowing limited angular movement.
